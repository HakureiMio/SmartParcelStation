# 低功耗策略说明

CR2450 纽扣电池容量有限，设计目标是事件驱动、短时工作、快速回到低功耗。

## 1. 基本策略

- 空闲状态尽快进入低功耗。
- RGB 与蜂鸣器只在必要场景短时启用。
- BLE 在满足交互的前提下使用较低广播/连接负载。
- 避免高频周期任务，改为事件触发。

## 2. 外设功耗控制

- RGB 使用低占空比闪烁，不常亮高亮度。
- 蜂鸣器模式设置最大持续时间，超时自动停止。
- 电池分压电路可通过 GPIO 使能，仅采样窗口打开。

## 3. 电池检测策略

- 默认不持续检测。
- 支持每日一次定时检测。
- 支持接收 `battery_check` 命令后立即检测。
- 电量等级变化或低电时触发事件上报。

## 4. 数据与隐私边界

- 夹具端不保存用户身份、包裹详情等隐私数据。
- 仅保留设备运行必要状态，降低存储与泄露风险。
